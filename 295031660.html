New IT Delivery Patterns : AWS Analytics 2: Marketing Analytics with Quicksight

Created by Karina Jonane, last modified by Thomas R. Evans on Apr 30, 2021

This Data-driven, cloud-powered asset gives Consulting resources the tools to load client data onto Amazon Web Services (AWS) in a secure S3 storage space, and then leverage a set of analytical and visualization tools to automatically format and analyze the data in real-time, on the fly.  This toolkit can be augmented with a variety of plug-and-play services available on AWS, either AWS created (QuickSight, etc.) or third party (Hadoop, Splunk, etc.).  These tools can make engaging clients in discussion of their analytics needs easier and can also open up broader discussions on cloud capabilities and cloud architectures.

AWS Architecture: Queries Against an Amazon S3 Data Lake

This exercise highlights that a streamlined flow (illustrated end-to-end below with check marks) through the following Amazon services can render powerful Business Intelligence:

You will be utilizing the following Amazon Web Services:

  • Amazon S3 - Simple Storage Service, here serving as an intermediate data storage container. More...
  • AWS Glue - a fully managed service that makes it simple to categorize, clean, enrich, and move data reliably between data stores. More...
  • AWS Glue Data Catalog - is an index to the location, schema, and runtime metrics of your data. The information there is used to create and monitor your ETL jobs. More...
  • AWS Glue Crawlers - as the method by which to populate the AWS Glue Data Catalog with tables. More...
  • AWS Athena - a serverless interactive query service that makes it easy to analyze data in S3 using standard SQL. More...
  • AWS QuickSight - a fast, easy-to-use business analytics service that makes it easy to visualize and extract business insights from data. More...
